# Isolation (2024)

A survival-based Unity game I developed during my 9th grade summer.

## Overview

Isolation is a small survival and crafting game where players must gather resources, craft tools, and survive against hostile enemies. The project was originally planned to be expanded further, but development was later discontinued.

## Controls

| Key | Action |
|------|--------|
| W, A, S, D | Move |
| E | Craft / Interact |
| Left Mouse Button | Attack |
| H | Heal |

## Objective

Defeat the Hyena and survive the environment.

## Features

### Resource Gathering & Crafting
- Collect resources throughout the world.
- Craft tools and items to aid your survival.

### Inventory System
- Obtain a backpack to store and manage items.
- Inspired by inventory systems found in survival games such as *The Last of Us*.

### Craftable Items

**Axe**
- Used to chop down trees for wood.
- Requires stone to craft.
- Has limited durability.

**Med Kit**
- Restores player health.
- Crafted using wood and pills.

## Development Notes

Isolation was one of my early Unity projects and served as an opportunity to explore game development concepts such as:
- Combat systems
- Crafting mechanics
- Resource gathering
- Inventory management
- Basic enemy AI

Although the project was never fully expanded beyond its original scope it was a valuable learning experience that helped develop my programming and game design skills.
