## Isolation


A survival-based Unity game developed during my 9th grade summer.

---

## Overview
Isolation is a survival and crafting game where players must gather resources, craft tools, and survive hostile enemies. The project was originally intended for further expansion but was later discontinued after the initial development phase.

---

## Controls

| Key | Action |
|-----|--------|
| W, A, S, D | Move |
| E | Craft / Interact |
| Left Mouse Button | Attack |
| H | Heal |

---

## Objective
Defeat the Hyena and survive the environment.

---

## Features

### Resource Gathering & Crafting
Players can collect resources throughout the world and craft tools and items to aid survival.

### Inventory System
A backpack-based inventory system allows players to store and manage items.  
Inspired by survival game inventory systems such as *The Last of Us*.

### Craftable Items

**Axe**
- Used to chop trees for wood  
- Requires stone to craft  
- Has limited durability  

**Med Kit**
- Restores player health  
- Crafted using wood and pills  

---

## Development Notes
Isolation represents one of my early Unity projects and was developed as an opportunity to explore core game development systems, including:

- Combat systems  
- Crafting mechanics  
- Resource gathering  
- Inventory management  
- Basic enemy AI  

Although the project was not expanded beyond its original scope, it provided valuable experience in programming and game design fundamentals.
